Tools

Tools I Use

Hardware

  • Macbook Pro (13.3”, Retina Display, mid 2020) - This laptop works well with all of the programs I frequently use, and I prefer the UI of Apple products. I also find it to be very compact and easy to carry around, despite not using the Macbook Air model.
  • Airpods (mid 2019) - I don't work with audio in a work setting, moreso in a hobby setting, so I don't need anything special. These do the job and are very easy to carry.
  • iPad Pro (12.9”, Retina Display, late 2015) - I adore the large size of this screen! I use my iPad mainly for photo editing and other small tasks.
  • Casio Graphing Calculator (FX-9750GII) - The Casio format is the first calculator format I became familiar with, which is why I work with this more often than Texas Instruments.

Development

  • Visual Studio Code - I prefer this editor above all because this was the first editor I learned to use and I prefer the set-up. I've listed below the extensions I use most.
    • CSS Formatter, HTML Preview, HTML CSS Support
    • Babel Javascript
    • Ruby, Rails

Services/Apps

  • Github - I typically code on VSCode and transfer the files into a repository on here. I find it to be a great source for storage of my projects.
  • Repl - During quarantine, I became very interested in web development and design. This website eases my ability to code and source a website of my own, and collaborate with others on a website as well. Highly recommend when experimenting with HTML/CSS/JS!
  • EarSketch - I use this website as a hobby, but I find coding on here to be very entertaining. I like remixing my favorite songs, or mixing instruments of genres that are not normally associated together. This site is a really good stress reliever!
  • NASA’s Eyes - This program is a very interesting and useful interactive that helps a lot when I do space-related research projects for school. I used this tons in my pre-requisite course for my NASA SEES internship.
